Esta carpeta contiene los archivos de instalación de los diferentes programas.

Como los archivos de programa ocupan mucho, no se incluyen aqui (se incluye un fichero con el mismo nombre pero en
realidad es un fichero que ocupa 0Kb.

El tamaño recomentado para comprimir los archivos de programa (WAS, eclipse, etc) son 200MB (209715200 bytes).
